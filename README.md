# 国民经济行业分类查询系统

基于 2017 年国民经济行业分类国家标准（GB/T 4754-2017）的在线查询工具。

## 功能特点

- **树状层级展示**：完整的 4 级分类体系（门类 → 大类 → 中类 → 细类）
- **全文搜索**：支持按代码、名称、详细说明搜索，Enter 键触发
- **分类统计**：共收录 **1382** 个行业细类，覆盖全部 20 个门类
- **详情展示**：点击展开查看详细说明，包含"包括"和"不包括"分类
- **响应式设计**：适配桌面和移动设备

## 目录结构

```
industrial_classifi/
├── index.html              # 前端页面
├── data/
│   └── industry_tree.json  # 行业分类数据（自动生成）
├── scripts/
│   └── parse_ref.py       # Excel 解析脚本
└── ref.xlsx               # 原始数据文件（需准备）
```

## 使用方法

### 方式一：直接打开（推荐）

```bash
cd industrial_classifi
python3 -m http.server 8080
# 访问 http://localhost:8080/
```

### 方式二：双击打开

直接双击 `index.html` 打开（部分浏览器可能限制本地 JSON 加载）

## 数据说明

### 层级结构

| 层级 | 示例 | 说明 |
|------|------|------|
| 门类 | A、B、C...T | 20 个大类 |
| 大类 | 01、72、85... | 97 个中类 |
| 中类 | 011、728... | 473 个中类 |
| 细类 | 0111、7281... | 1382 个小类 |

### 描述格式

- `◇` - 表示包含范围
- `◆` - 表示不包括范围
- `—` - 子项列表

### 示例

```
◇ 包括对下列农业种植活动：
— 稻谷种植
— 小麦种植
— 玉米种植
◆ 不包括：
— 水果种植（列入水果种植相关类别）
```

## 数据更新

如需从原始 Excel 更新数据：

```bash
# 确保有 ref.xlsx 文件
python3 scripts/parse_ref.py
```
